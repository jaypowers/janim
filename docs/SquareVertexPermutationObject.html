Class: SquareVertexPermutationObject

SquareVertexPermutationObject(optionsopt)

Scene-local square graphic that moves vertex labels according to a permutation. This is used in the `D_4 -> S_4` scene to show the same queued maps on both the square and the two-line permutation display.
