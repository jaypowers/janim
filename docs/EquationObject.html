Class: EquationObject

EquationObject(latex, optionsopt)

Renders a LaTeX expression into a p5 image and animates it. Use `Equation(...)` for mathematical labels, notation, and titles that should be typeset by MathJax.

Example
const title = new Equation("\\text{Permutations}", {
  x: width / 2,
  y: 72,
  size: 28,
  color: color(220, 235, 255)
});

title.reveal({ duration: 1.5, letters: true });
