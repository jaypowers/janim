Class: SymmetricGroupObject

SymmetricGroupObject(n, optionsopt)

Animated display for the symmetric group `S_n`. `SymmetricGroupObject` shows a permutation as a function from an input row to an image row. It is useful for contrasting `S_n`, which contains every possible rearrangement of `n` labels, with a subgroup such as `D_4`, which contains only the rearrangements coming from rigid square motions. Commands queued with `push(...)`: - `"pause", seconds` holds the current permutation. - `"map", [2, 1, 3, 4]` animates directly to a chosen image row. - `"swap", 1, 2` swaps two positions. - `"cycle", [1, 2, 3, 4]` cycles the listed positions.

Example
const s4 = SymmetricGroup(4, {
  x: width / 2,
  y: height / 2,
  title: "S_4 contains every rearrangement"
});

s4.push("swap", 1, 2);
s4.push("cycle", [1, 2, 3, 4]);
s4.run({ loop: true });
