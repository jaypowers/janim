Class: SceneQueueObject

SceneQueueObject()

Minimal scene sequencer for p5 sketches. A scene is any object with optional `setup()`, `update(dt)`, `draw()`, and `resize()` methods. `SceneQueueObject` lets sketches keep each topic in its own object, then move through those scenes with `next()` and `previous()`.

Example
const scenes = SceneQueue();
scenes.push(createDihedralScene());
scenes.push(createSymmetricScene());
scenes.run();

function draw() {
  background(5, 16, 32);
  scenes.update();
  scenes.draw();
}
