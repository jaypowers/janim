Class: PermutationObject

PermutationObject(elements, optionsopt)

Animated display for a finite permutation of a small ordered set. A `PermutationObject` shows three related ideas at once: a set such as `{1, 2, 3}`, an ordered tuple such as `(2, 3, 1)`, and the standard two-line permutation notation rendered with LaTeX. In two-line notation, the top row is the input and the bottom row is the image. For example, `\begin{pmatrix}1 & 2 & 3\\2 & 3 & 1\end{pmatrix}` means `1 -> 2`, `2 -> 3`, and `3 -> 1`.

Example
const p = new Permutation([1, 2, 3], {
  x: width / 2,
  y: height / 2,
  title: "All six permutations"
});

p.push("pause");
p.push("map", [1, 3, 2]);
p.push("pause");
p.push("swap", 1, 3);
p.run({ loop: true });

push(command, …args) → {PermutationObject}

Adds either a command or a floating text note. Commands: - `"pause"` pauses on the current permutation. - `"map", [a, b, c]` animates to a specific order. - `"swap", 1, 3` swaps two 1-based tuple positions using a circular path. - `"rotate"` cyclically rotates the tuple. - `"flip"` reverses the tuple. Any other string is treated as text attached to this object.
