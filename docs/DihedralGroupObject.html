Class: DihedralGroupObject

DihedralGroupObject(n, …args)

Animated regular polygon model for the dihedral group `D_n`. The object draws an `n`-gon and animates rotations, flips, scaling, and translations. It can be driven immediately with methods such as `rotate()`, or with queued commands through `push(...)` and `run(...)`.

Example
const d4 = DihedralGroup(4, {
  x: width / 2,
  y: height / 2,
  radius: 90,
  captionLatex: "D_{4}:\\text{ rotations and flips}"
});

d4.push("rotate");
d4.push("flip", 1);
d4.push("unflip", 1);
d4.run({ loop: true });
