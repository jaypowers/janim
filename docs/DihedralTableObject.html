Class: DihedralTableObject

DihedralTableObject(n, …args)

Animated multiplication table for a dihedral group `D_n`. The table lays out rotations first, then reflections, and renders the cell entries with MathJax. It can reveal the grid, reveal entries, and highlight one multiplication lookup through queued `push(...)` commands.

Example
const table = DihedralTable(3, {
  x: 48,
  y: 150,
  cellSize: 44,
  heading: "D_3 multiplication"
});

table.push("revealGrid");
table.push("revealElements");
table.push("highlight", "r", "sigma");
table.run();
